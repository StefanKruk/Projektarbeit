%Befehle für Glossar
\newglossaryentry{glos:Docker}{
    name=Docker,
    description={Docker ist ein Werkzeug, um Anwendungen mit Hilfe von Betriebssystemvirtualisierungen zu isolieren. Dafür werden sogennante Docker-Images genutzt, in denen die virtuelle Umgebung definiert und konfiguriert wird. Dabei kann eine Anwendung automatisch Installiert und gestartet werden. Docker arbeitet dabei auf Basis eines Linux Kernels. Anders als bei virtuellen Maschinen, welche zum Beispiel mit Hilfe von VirtualBox erstellt werden können, ist eine vollständige Isolation des Systemes nicht möglich. Alle Docker-Images auf einem System, teilen sich den Linux Kernel des zugrunde liegenden Systems. Im Falle von Windows wird dafür eine virtuelle Maschine, mit einem Linux Kernel gestartet, auf welchem die Images laufen.}
}
\newglossaryentry{glos:Vagrant}{
    name=Vagrant, 
    description={Vagrant ist ein Werkzeug zum erstellen von Standardisierten virtuellen Maschinen mit Hilfe von VirtualBox oder VMWare. Neben dem erstellen kann man mit Hilfe von Vagrant die virtuellen Maschinen steuern. In Kombination mit Docker, ist es ein Mächtiges Werkzeug zum einfachen erstellen von Systemen}
}
